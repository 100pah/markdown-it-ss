
## Install
```shell
code ~/.bash_profile
```
Add:
```shell
alias mditss="${USER_BASE}/your/path/to/markdown-it-ss/index.js"
```
Then:
```shell
source ~/.bash_profile
```

## Build

See `package.json`.

Change input path in `index.js`

## Usage
+ Compile
    ```shell
    mditss some.md
    ```
+ Help
    ```shell
    mditss --help
    ```
